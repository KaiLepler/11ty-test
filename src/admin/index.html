Publishing a Draft

To publish a draft, follow these steps:

  1. Move the draft file from the src/drafts/ folder to the src/posts/ folder.
  2. Edit its frontmatter:
    • Remove the draft: true line (or set it to draft: false).
    • Ensure the date field is set to the desired publication date.
